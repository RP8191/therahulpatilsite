# When ChatGPT Keeps Changing Its Answer: A Real Conversation Log

I recently had an interaction with ChatGPT that made me pause and reflect on how much I trust AI-generated answers—especially for factual, detail-heavy queries.

It started with a simple question:

> List all the songs sung together by Kumar Sanu and Lata Mangeshkar.

---

## The Initial Answer

ChatGPT gave me a list that included:

```

Tujhe Dekha To Yeh Jaana Sanam
Saathiya Tune Kya Kiya

```

It also included some uncertain or loosely attributed entries.

---

## First Refinement

When I asked for a **verified discography**, the list was reduced to:

```

Saathiya Tune Kya Kiya
Tujhe Dekha To Yeh Jaana Sanam

```

So far, things seemed to be getting more precise.

---

## Introducing Another Song

I then asked about:

```

Madhosh Dil Ki Dhadkan

```

ChatGPT confirmed:

```

Kumar Sanu + Lata Mangeshkar

```

And updated the total count of songs accordingly.

---

## The Turning Point: “O Jaana Na Jaana”

I brought up another song:

```

O Jaana Na Jaana

```

### Response 1
ChatGPT said:

```

Kumar Sanu + Alka Yagnik

```

And clearly stated Lata Mangeshkar was not part of the song.

---

### I Pushed Back

I responded that it was sung by Kumar Sanu and Lata Mangeshkar.

---

### Response 2
ChatGPT doubled down:

```

Kumar Sanu + Alka Yagnik

```

It referenced “official soundtrack credits” and multiple databases.

---

### I Asked for Sources

It listed sources like:

- Audio cassette/CD credits  
- Film credits  
- Music label (Tips)  
- Discogs, IMDb, Wikipedia  

---

### Response 3 (First Flip)

ChatGPT reversed its answer:

```

Kumar Sanu + Lata Mangeshkar

```

It acknowledged the earlier answer was wrong.

---

### Response 4 (Second Flip)

It changed again:

```

Kumar Sanu + Alka Yagnik

```

This time stating no primary source supports Lata’s involvement.

---

### Response 5 (Third Flip)

It flipped once more:

```

Kumar Sanu + Lata Mangeshkar

```

---

## At This Point

I pointed out that it had:

```

flip flopped 4 times

```

---

## ChatGPT’s Acknowledgement

To its credit, ChatGPT acknowledged:

- It had been inconsistent  
- It relied on pattern-based recall  
- It corrected itself without proper re-verification  

It explicitly stated that its earlier responses on this topic should be considered:

```

unreliable

```

---

## What I Took Away

This wasn’t a complex philosophical debate or an ambiguous topic. It was a **specific factual question** about song credits.

Yet the answers:

- Changed multiple times  
- Were given with confidence each time  
- Contradicted each other  

---

## Final State

No stable answer was maintained for:

```

O Jaana Na Jaana

```

And the assistant itself acknowledged the inconsistency.

---

## Closing Thought

This experience didn’t come from a vague or subjective question. It came from a **simple, verifiable query**.

That’s what made it worth documenting.
